#  ASL to Bisaya Translation Glove

A wearable glove that translates American Sign Language (ASL) hand gestures into spoken Bisaya using an ESP32, flex sensors, and motion sensing.

## Project Status

 Current Stage: Prototype Development (40%)

Progress:
- ✅ Research completed
- ✅ Materials acquired
- ✅ Initial circuit designed
- 🔄 ESP32 programming
- ⏳ Gesture recognition
- ⏳ Audio output
- ⏳ Field testing

## Current Hardware

- 2 ESP32
- 5 Flex Sensors
- 2 MPU6050
- DFPlayer Mini
- Speaker
- Li-ion Battery

## Goals

- Detect ASL alphabet
- Convert gestures into Bisaya speech
- Portable and rechargeable
- Low-cost educational device
